##### 53 中等
```
function test(foo) {
    console.log(foo.n);
    var foo = {n:2};
}
test(foo); // foo is undefined(foo指的是函数调用时传的参数foo)
test({n:1}); // 1  会生成：var foo = 你传的值   变量存在提升，但是如果变量名称跟已经声明的形参或函数相同，则变量声明不会干扰已经存在的这类属性
```
